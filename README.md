## Setup
### Run
Run `npm start` to start a webserver and the Bare server. You may deploy Google by using an external bare server and a static file host. Google must be built before attempting to start.

### Build
Run `npm run build` to build app for production to the `build` folder.

### Development
Run `npm run dev` to run the app in development mode.

### Deploy
Click a button at the top of this page and follow the directions for an easy way to host Google.
